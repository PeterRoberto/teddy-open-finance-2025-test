# Teddy Open Finance 

Teste para a empresa [Teddy Open Finance](https://teddydigital.io/).
<br>
<br>


## React + Node + Vite + TypeScript + TailwindCSS

Para esse projeto utilizei:
* React v19.1.1
* Node 22.19.0
* Vite v7.1.2
* Typescript 5.8.3 
* TailwindCSS 4.1.12
<br>


## Instalar pacotes

* Na pasta raiz rode
```bash
npm install 
```
<br>


## Rodar a aplicação

* Na pasta raiz rode

```bash
npm run dev 
```

### Rodar com Docker
```bash
docker build -t teste-cliente .
docker run -p 3000:80 teste-cliente
