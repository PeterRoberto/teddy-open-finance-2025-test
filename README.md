# Teddy Open Finance 

Teste para a empresa [Teddy Open Finance](https://teddydigital.io/).


# React + Node + Vite + TypeScript + TailwindCSS

Para esse projeto utilizei:
* React v19.1.1
* Node 22.19.0
* Vite v7.1.2
* Typescript 5.8.3


## Instalar pacotes

```bash
npm install 
```
na pasta raiz


## Rodar a aplicação

```bash
npm run dev 
```
na pasta raiz
